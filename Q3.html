Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented, and functional programming.

Why Choose Python? Python is ideal for beginners and experts alike, offering a gentle learning curve while remaining powerful enough for complex applications.

Getting Started

To start programming in Python, you'll need to install Python on your computer. Python can be downloaded from the official website at python.org. Most modern operating systems come with Python pre-installed, but it's recommended to install the latest version.

You can verify your Python installation by opening a terminal or command prompt and typing:

python --version

Your first Python program is traditionally a "Hello World" program. Here's how simple it is:

print("Hello, World!")

Python files use the .py extension. You can run a Python file from the command line using the python command followed by the filename.

  • Install Python from python.org
  • Choose an IDE or text editor (VS Code, PyCharm, Sublime Text)
  • Write your first program
  • Run it using the terminal or your IDE's run button
Variables and Data Types

In Python, variables are created when you assign a value to them. Python is dynamically typed, so you don't need to declare the variable type explicitly.

Here are examples of different data types:

name = "Alice"        # String
age = 25              # Integer
height = 5.6          # Float
is_student = True     # Boolean

Python has several built-in data types including integers, floats, strings, booleans, lists, tuples, dictionaries, and sets. Understanding these types is fundamental to Python programming.

You can check the type of any variable using the type() function:

print(type(name))     # Output: <class 'str'>
  • Integers: whole numbers like 42, -17, 0
  • Floats: decimal numbers like 3.14, -0.5, 2.0
  • Strings: text enclosed in quotes like "Hello" or 'Python'
  • Booleans: True or False values
Control Flow

Control flow statements allow you to control the execution of your code based on conditions. The most common control flow statements in Python are if, elif, and else.

age = 18

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

Python also supports loops for repetitive tasks. The two main types of loops are for loops and while loops.

Here's a simple for loop example:

for i in range(5):
    print(f"Count: {i}")

And a while loop example:

count = 0
while count < 5:
    print(count)
    count += 1
  • Use if statements for conditional execution
  • Use for loops to iterate over sequences
  • Use while loops when you need to repeat until a condition is met
Functions

Functions are reusable blocks of code that perform a specific task. In Python, you define a function using the def keyword.

def greet(name):
    return f"Hello, {name}!"

message = greet("Alice")
print(message)  # Output: Hello, Alice!

Functions can take multiple parameters and return values. They help organize your code and make it more maintainable.

You can also define functions with default parameter values:

def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

print(greet("Bob"))              # Output: Hello, Bob!
print(greet("Bob", "Hi"))        # Output: Hi, Bob!

Functions are a fundamental building block in Python programming, allowing you to write modular and reusable code.

Lists and Dictionaries

Lists and dictionaries are two of the most commonly used data structures in Python. Lists are ordered collections of items, while dictionaries store key-value pairs.

Creating and using lists:

fruits = ["apple", "banana", "cherry"]
print(fruits[0])       # Output: apple
fruits.append("date")  # Add an item
print(len(fruits))     # Output: 4

Creating and using dictionaries:

person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}
print(person["name"])  # Output: Alice
person["email"] = "alice@example.com"  # Add a new key-value pair

Both lists and dictionaries are mutable, meaning you can modify them after creation. They're incredibly versatile and used in almost every Python program.

  • Lists use square brackets [] and maintain order
  • Dictionaries use curly braces {} and store key-value pairs
  • You can nest lists and dictionaries inside each other
  • List indices start at 0 in Python
